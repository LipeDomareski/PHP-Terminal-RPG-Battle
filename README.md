# PHP Terminal RPG Battle

Mini RPG de batalhas em terminal desenvolvido em PHP para praticar:

- Programação Orientada a Objetos
- Herança
- Polimorfismo
- Classes Abstratas
- Factory Pattern

## Como executar

php index.php

## Personagens

- Warrior
- Knight
- Sorcerer
- Cleric
- Hunter
- Thief

## Objetivo

Projeto criado para praticar conceitos de POO e arquitetura de sistemas em PHP.
